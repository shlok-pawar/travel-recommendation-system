# Travel Recommendation System

This project recommends travel destinations based on user preferences, including location, place type, and interests.

## Features
- Suggests destinations based on data
- Uses a real dataset of Indian cities and places
- Simple and efficient logic

## Tech Stack
- Python (streamlit)
- Excel datasets

## How to Run
1. Download the files
2. Run main.py
3. While running main.py, make sure you have Python and its libraries installed.
4. If not, install them. And run using "python -m streamlit run main.py".
