# Smart Travel Recommendation System

A web-based travel recommendation system built using Streamlit that provides personalized destination suggestions based on user preferences such as location, budget, duration, travel type, and season.

## Features
- Filter-based travel recommendations
- Distance calculation using the Haversine formula
- Supports multiple travel categories (solo, family, friends, couples)
- Budget and duration-based filtering
- Displays structured output with:
  - Destination name
  - Distance (km)
  - Estimated budget
  - Best season
  - Group suitability
- Optional map visualization of destinations

## Tech Stack
- Python
- Streamlit
- CSV Dataset
- Geolocation (Haversine Formula)

## How It Works
Users input their preferences, and the system filters destinations from a dataset based on distance and criteria, providing relevant travel suggestions instantly.

## Future Improvements
- Integration with real-time travel APIs
- User login and saved preferences
- AI-based recommendation system

## How to Run
1. Download the files
2. Run app.py
3. While running main.py, make sure you have Python and its libraries installed.
4. If not, install them. And run using "python -m streamlit run app.py".
